# 01-1 인공지능과 머신러닝, 딥러닝

## 인공지능이란

인공지능은 사람처럼 학습하고 추론할 수 있는 지능을 가진 컴퓨터 시스템을 만드는 기술
    - 분류
            약인공지능 : 특정 분야에서 사람의 일을 도와주는 보조 역할만 가능
            강인공지능(인공일반지능) : 사람과 구분하기 어려운 지능을 가진 컴퓨터 시스템
            
## 머신러닝이란
    
머신러닝이란 규칙을 모두 프로그래밍 하지않아도 자동으로 데이터에서 규칙을 학습하는알고리즘을 연구하는 분야이다. 인공지능 하위분야 중에서 지능을 구현을 위한 소프트웨어를 담당하는 핵싱 분야이다. 머신러닝은 통계학과 깊은 관련이 있다. 하지만 최근 신러닝의 발전은 통계,수학이론보다 경험을 바탕으로 발전하기도 한다. 컴퓨터 과학 분야가 이것을 주도한다. 그리고 이분야의 대표적인 머신러닝 라이브러리는 사이킷런이다. 사이킷런은 파이썬을 사용하여 간편하고 배우기 쉬운장점이 있어 대부분의 사람들이 사이킷런을 사용하고있다.

## 딥러닝이란

딥러닝이란 여러 머신러닝 알고리즘 중 인공 신경망을 기반으로 한 방법들을 통칭하는 말이다.

# 01-1 인공지능과 머신러닝, 딥러닝

## 인공지능이란

인공지능은 사람처럼 학습하고 추론할 수 있는 지능을 가진 컴퓨터 시스템을 만드는 기술
    - 분류
            약인공지능 : 특정 분야에서 사람의 일을 도와주는 보조 역할만 가능
            강인공지능(인공일반지능) : 사람과 구분하기 어려운 지능을 가진 컴퓨터 시스템
